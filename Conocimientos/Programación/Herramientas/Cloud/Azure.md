Microsoft Azure es una plataforma de computación en la nube pública de Microsoft que permite varios productos/servicios como almacenamiento, redes, IA, bases de datos, etc...

Permite crear, ejecutar y administrar aplicaciones en entornos híbridos , en la nube o perimetrales, escalando según el usuario necesite y se paga por uso.

Aspectos Clave:

- Servicios principales: Incluye infraestructura como servicio (IaaS), Plataforma como servicio (PaaS) y Software como servicio (SaaS) destacando la integración con herramientas como Windows Server, SQL Server y Active Directory.

- Seguridad y flexibilidad: Ofrece alta seguridad, cumplimiento normativo y soporte para numerosos lenguajes de programación y marcas de trabajo.

- IA y Datos: proporciona bases de datos modernas, análisis de datos avanzado e inteligencia artificial para soluciones innovadoras.

- Red global: Funciona sobre una red masiva de centro de datos de Microsoft.
