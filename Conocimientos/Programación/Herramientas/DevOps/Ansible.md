Ansible es una herramienta de automatización que permite configurar sistemas, implementar software y orquestar flujos de trabajo de forma sencilla, sin agentes (agentless) y utiliza el lenguaje YAML que tiene la extensión .yml

Características:

- Agentless: No requiere instalar software adicional, al instalar ansible ya se puede utilizar utilizando SSH en linux y WinRM en Windows.

-  YAML: Utiliza archivos llamados playbooks escritos en YAML.

- Idempotencia: Garantiza el perfecto estado del sistema aunque se ejecute varias veces una función, acción u operación sin causar efectos secundarios.

- Módulos: Emplea módulos para tareas específicas como instalar paquetes, gestionar usuarios o configurar archivos.

- Infraestructura como código (IaC): Permite definir la estructura como código facilitando el control de versiones y la replicación de entornos.
