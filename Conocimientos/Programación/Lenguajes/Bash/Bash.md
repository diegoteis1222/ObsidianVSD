Bash es un intérprete de comandos y lenguaje de programación de shell, utilizado en sistemas Linux y Unix de forma estándar, que actúa como interfaz entre el usuario y el sistema operativo. Permite ejecutar tareas repetitivas mediante scripts (.sh) y gestionar archivos o procesos, siendo esencial para la administración de sistemas.

Aspectos claves: 

- Intérprete de comandos: Lee comandos introducidos por el usuario e una terminal o desde un archivo y los ejecuta.

- Scripting: capacidad para agrupar comandos en un archivo (script) y ejecutarlos a la vez para automatizar procesos complejos.

 - Características: Soporta variables, bucles, estructuras condicionales y funciones.

-  Uso: Es el shell predeterminado en la mayoría de distribuciones Linux y macOS.

-  Funcionalidad: Actúa como un entorno de programación para interactuar con el sistema operativo.
